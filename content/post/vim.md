## ノーマルモード

### 再描画

| コマンド | 説明 |
| --- | --- |
| `zt` | 現在行を画面の１行目にする |
| `zz` | 現在行を画面の中央にする |
| `zb` | 現在行を画面の最下行にする |

## 挿入モード

| コマンド | 説明 |
| --- | --- |
| `<C-o>` | １つコマンドを実行して挿入モードに戻る |
| `<C-r>{register}` | 指定したレジスタの内容を挿入する |
